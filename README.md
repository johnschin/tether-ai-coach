# Tether AI Coach

B2B corporate psychological resilience coaching platform.
Built on Claude API, Supabase, Cloudflare Workers, and Netlify.

## Architecture
- **AI:** Anthropic Claude API via Cloudflare Worker proxy
- **Database:** Supabase (project: tether-ai-coach)
- **Auth:** Supabase Email Auth
- **Hosting:** Netlify
- **Framework:** PWA (Progressive Web App)

## Key Features
- ADKAR-informed change readiness coaching
- Persistent memory across sessions
- Three-tier crisis escalation protocol
- Anonymized HR analytics dashboard
- Zero individual data exposure to employers

## Stack
- worker/ — Cloudflare Worker (proxy + API logic)
- app/ — PWA front end
- supabase/migrations/ — Database schema
